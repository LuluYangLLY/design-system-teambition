## Icon Size

Default icon size is `16px * 16px`. But we can get other size by pass `size` in.

<Preview>
    <Icon icon='warning' size='20'></Icon>
    <Icon icon='star' size='50'></Icon>
    <Icon icon='plus' size='100'></Icon>
</Preview>

## Icon Color

Default icon color is `#333333`. But we can get other color icons by pass `color` in.

<Preview>
    <Icon icon='remove' color='red' size='20'></Icon>
    <Icon icon='star' color='purple' size='20'></Icon>
    <Icon icon='plus' color='#1EA7FD' size='20'></Icon>
</Preview>
